# Milestone 1 — MVP (Complete)

Core reading experience with AI integration. Shipped as v0.1–v0.2.x.

## What's included
- EPUB reading with paginated/scrolling view (foliate-js)
- Library management (import, delete, collections)
- Reading progress tracking
- AI chat assistant (contextual Q&A, passage explanation)
- AI quick lookup (definition + context on word selection)
- Highlights & bookmarks with color/note management
- Vocabulary management with spaced repetition
- Settings (AI provider, model, theme, reading preferences)
- Multi-provider AI support (Anthropic, OpenAI-compatible, Ollama, MiniMax)
- OpenAI OAuth login
- iCloud sync (books, DB, settings via relative paths)
- PDF support

*Feature specs for Milestone 1 have been removed — the shipped code is the source of truth.*
